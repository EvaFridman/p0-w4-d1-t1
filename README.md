# p0-w4-d1-t1 · Шахматная доска из массивов

> Тип: объёмная (генерация структуры методами + DOM) · Репозиторий: один отдельный

> Подключить `.js` к HTML через `<script>`, отсмотреть в браузере.

## Задача

Сгенерировать шахматную доску целиком из данных: не верстать 64 клетки руками, а построить их методами массивов и отрисовать в DOM. Чередование цветов и стартовая расстановка фигур вычисляются, а не прописываются вручную.

```
сгенерировать доску 8×8 методами, не руками
┌──────────────────────────────────────────────────────┐
│ Array.from → ряды и клетки                           │
│ цвет клетки: (row + col) % 2                         │
│ фигуры стартовой позиции из массива                  │
│ рендер в DOM (grid 8×8)                              │
└──────────────────────────────────────────────────────┘
белые/чёрные чередуются, фигуры на местах
```

## Дано

```html
<div id="board"></div>
```

```css
#board { display: grid; grid-template-columns: repeat(8, 48px); }
.cell { width: 48px; height: 48px; display: grid; place-items: center; font-size: 28px; }
.light { background: #f0d9b5; }
.dark  { background: #b58863; }
```

```js
// стартовая линия фигур (слева направо), 2-я строка — пешки
const backRank = ["♜","♞","♝","♛","♚","♝","♞","♜"];
```

## Что нужно сделать (по релизам)

**Release 1 — пустая доска.** Через `Array.from` построить 8 рядов по 8 клеток (можно вложенным `Array.from` или `map`). Цвет клетки определить по чётности `(row + col) % 2` и повесить класс `light`/`dark`. Отрисовать в `#board`. Ни одного `for`.

**Release 2 — фигуры.** На верхние две строки поставить чёрные фигуры (0-я — `backRank`, 1-я — восемь пешек `♟`), на нижние две — белые (`♙` и белый `backRank` `♖♘♗♕♔♗♘♖`). Символ фигуры класть в текст клетки. Расстановку взять из массивов, а не расписывать по клеткам.

**Release 3 — координаты.** Добавить подписи: снизу буквы `a–h`, слева цифры `8–1`. Сгенерировать их методами (`Array.from`/`map`), а не хардкодом восьми подписей.

## Ожидаемый результат

Доска 8×8 с правильным чередованием цветов (левая верхняя клетка — светлая по стандарту при таком правиле — договорись сама и будь последовательна), чёрные фигуры сверху, белые снизу, координаты по краям. Всё построено из массивов.

## Критерии приёмки

* Доска и фигуры сгенерированы методами массивов (`Array.from`/`map`), без `for` и без ручной верстки 64 клеток.
* Цвет клетки вычисляется по `(row + col) % 2`; фигуры расставлены из массивов.
* Координаты `a–h` / `8–1` сгенерированы, а не прописаны вручную.
